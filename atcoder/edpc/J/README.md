式変形する。頻出。

```
// f(3個載ってる皿の数,2個載ってる皿の数,1個載ってる皿の数) := 期待値
寿司のある皿の数 := 3個載ってる皿の数+2個載ってる皿の数+1個載ってる皿の数

f(皿3,皿2,皿1) =
    f(皿3-1,皿2,皿1) * (皿3 / (寿司のある皿の数 / 皿の数)) +
    f(皿3,皿2-1,皿1) * (皿2 / (寿司のある皿の数 / 皿の数)) +
    f(皿3,皿2,皿1-1) * (皿1 / (寿司のある皿の数 / 皿の数)) +
    f(皿3,皿2,皿1)   * (1.0 - (寿司のある皿の数 / 皿の数))

無限再帰しそうだけど式変形すれば良い。

1. f(皿3,皿2,皿1) - f(皿3,皿2,皿1) * (1.0 - (寿司のある皿の数 / 皿の数)) = (略)
2. f(皿3,皿2,皿1) * (1.0 - (1.0 - (寿司のある皿の数 / 皿の数))) = (略)
3. f(皿3,皿2,皿1) * (寿司のある皿の数 / 皿の数) = (略)
4. f(皿3,皿2,皿1) = (略) / (寿司のある皿の数 / 皿の数)

```
