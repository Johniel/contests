# TDPC F

`dp[今どこ][停止or通過]`

`dp[i][停止]` にk回連続で停止するパターン数は `dp[i - k][通過]` 。<br/>
※ `dp[i - (k - 1)][通過]` では何回連続で停止なのか分からないため。

```
辻褄を合わせるために通過を入れる。
dp[0][通過]=1
dp[0][停止]=0

dp[1][通過]=0
dp[1][停止]=1

dp[i][通過] = dp[i - 1][通過] + dp[i - 1][停止]
dp[i][停止] = dp[i - 1][通過] + dp[i - 1][停止] - dp[i - k][通過]
```
